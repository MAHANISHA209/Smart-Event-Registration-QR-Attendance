Smart Event Registration and QR Attendance System

**Overview**

Smart Event Registration and QR Attendance System is a web application that allows participants to register for events online and mark attendance using QR codes.
It reduces manual work, saves time, and provides accurate attendance records.

**Problem Statement**

Managing event registrations and attendance manually is time-consuming and error-prone. Traditional attendance methods using paper sheets can lead to duplicate entries, data loss, and long waiting times. Organizers need a secure and efficient system to automate registration and attendance tracking.

**Solution**

The Smart Event Registration and QR Attendance System allows participants to register online and receive a unique QR code. During the event, organizers scan the QR code to mark attendance automatically. The system stores attendance records in a database and generates reports instantly.

**Features**
